# 📊 Sales Data Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a sales dataset using Python. The objective is to understand the dataset, clean the data, identify trends, and visualize key business insights.

---

## 🚀 Features

- Load and inspect the dataset
- Data cleaning
- Handle missing values
- Remove duplicates
- Summary statistics
- Data visualization
- Identify trends and patterns

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Project Structure

```
DS-ML/
│── EDA_project.py
│── sales_data.csv
│── README.md
```

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/SharanyaDN-art/DS-ML.git
```

2. Move into the project folder

```bash
cd DS-ML
```

3. Install dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

4. Run the project

```bash
python EDA_project.py
```
